# Decision, Risk, and Reliability Atlas
[Live Site](https://robert-z-lehr.github.io/Glossary-DRR/)

A structured glossary atlas for core concepts in Decision Analysis, Risk, and Reliability engineering. This project provides a clean, expandable reference interface that can be hosted entirely on GitHub Pages. It includes:

[1] An editable concept glossary

[2] Structured concept fields

[3] A clean searchable table interface

[4] PDF export capability

[5] The ability to add new concepts dynamically

The long-term goal is to build a course-scale or field-scale atlas of primitives, similar to a technical encyclopedia but organized for rapid learning and reference.
